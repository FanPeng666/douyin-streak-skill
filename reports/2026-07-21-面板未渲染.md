## 三、根本原因分析

> **抖音网页版（"抖音精选" / Douyin Select）桌面端的私信面板在前端未渲染。**

具体表现：
1. **用户已登录**：Cookie 有效，头像显示，火花 API 可访问
2. **资源已加载**：点击"消息"后 `chat_fly` 模块和 IM APIs 全部正常调用
3. **面板未渲染**：虽然相关 DOM 元素（`#IMpushListBoxId`）存在，但 `visible=false`，且没有 UI 控件
4. **URL 路由受限**：`/messages`、`/im` 等直链均返回 404

这是抖音桌面 Web 端的已知特性 — 私信功能在桌面浏览器上仅提供 API 访问，不提供完整的 UI 体验（可能需要客户端 APP 或特定设备指纹）。
