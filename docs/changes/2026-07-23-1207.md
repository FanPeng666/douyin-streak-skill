## Hook 机制

- 事件：`Stop`（每次任务结束时触发一次）
- 类型：`command`，执行 `python ${CODEBUDDY_SKILL_DIR}\\scripts\\check_changelog.py`
- 作用：检测源码文件有修改时，检查 docs/changes/ 下是否有今天的记录 + CHANGELOG.md 是否已更新
- 无修改时静默退出（exit 0）
- 检测到遗漏时输出提醒（exit 1，但不阻断任务）

## 注意事项

- `python` 路径由 WorkBuddy 环境自动解析，无需写死
- `${CODEBUDDY_SKILL_DIR}` 指向项目根目录
- 检查脚本不依赖 git（避免沙箱环境下 git 不可用）
- `.workbuddy/` 被 `.gitignore` 排除，settings.json 和脚本不同步到远程，需要在本地单独维护
