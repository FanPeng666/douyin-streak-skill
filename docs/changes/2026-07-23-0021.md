## Prompt 变化

- 参数从 CLI 传入 → Prompt 文本传入（`Cookie：xxx` / `好友名称：xxx` / `发送内容：xxx`）
- 新增 `parsePromptArgs()` 解析函数
- 用户触发词后附参数即可，无需修改代码

## Skill 变化

- SKILL.md frontmatter: version 1.0.0 → 2.0.0
- 新增「Prompt 参数」章节和参数表
- 安全约定：Cookie 通过 Prompt 参数传入，不写入文件

## 技术决策

### 反检测
- UA 匹配 Chromium 版本（Chrome 127.0.0.0）
- 多维反检测：navigator.{webdriver,languages,plugins} + window.chrome

### DOM 交互
- **好友匹配**：class 选择器（已失效）→ TreeWalker 文本扫描
- **好友点击**：坐标偏移 → Playwright locator `text="好友名"`
- **私信 Tab**：新增 switchToPrivateMessageTab()，默认在「互动消息」需切「私信」
- **发送方式**：Enter 优先 + hover+click 发送按钮兜底
- **验证方式**：DraftEditor 选择器 + body 文本兜底

### 工程化
- `scripts/utils.js` 抽取通用工具函数
- `reports/` 和 `.workbuddy/` 加入 .gitignore

## 注意事项

- 抖音限制非互关用户每天仅 1 条消息
- DraftEditor 的 Enter 行为与发送按钮 click 不同（Enter 更可靠）
- node_modules 需 Junction 软链到 managed workspace
- 每次更新后需更新 CHANGELOG.md 和 docs/changes/
